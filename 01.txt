SELECT를 원하는 것을 출력할 때 입력된 순서로 출력됨
SELECT publisher : 중복 제거
WHERE(=,<>,<,<=,>,>=) : 연산기호 사용가능(콤마 사용 X)
WHERE(BETWEEN,AND) : 값의 범위 지정 가능
WHERE(IN) : 집합의 원소인지 판단
WHERE(LIKE) : 문자열 패턴 비교
-찾는 단어가 포함된 것을 찾고 싶다면 와일드 문자 '%'를 사용
-특정 위치에 한문자만 대신할 떄는 와일드 문자 '_'를 사용
SELECT "id" AS "아이디" - AS는 선택한 열의 이름을 바꿈
